🎮 VORTEX — Tactical Battle Arena Website

A visually immersive, high-performance gaming landing page designed for a fictional 5v5 tactical arena game.
Built with a strong focus on UI/UX design, animation, and futuristic aesthetics, this project showcases advanced frontend creativity using pure HTML, CSS, and JavaScript.

🚀 Features
⚡ Modern Hero Section
Animated grid background, glowing effects, and dynamic UI
Interactive 3D-style character illustration
Call-to-action buttons (Play / Watch Cinematic)
🎭 Agent Showcase
Multiple character cards with hover animations
Unique themes (Fire, Energy, Shield, etc.)
Smooth transitions and glow effects
🗺️ Battleground Section
Interactive map layout with zones and HUD-style UI
Animated pings and tactical elements
🏆 Ranking System
Tier-based ranking cards
Clean and structured competitive progression UI
⏳ Season Section
Countdown timer
Reward highlights and trophy visuals
🎨 Advanced UI Effects
Custom cursor & cursor ring
Scroll progress bar
Glassmorphism + neon glow effects
Smooth reveal animations
🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Design: Custom UI/UX (Gaming + Futuristic Theme)
Fonts: Google Fonts (Orbitron, Bebas Neue, Rajdhani)
Animations: CSS Keyframes + Transitions
